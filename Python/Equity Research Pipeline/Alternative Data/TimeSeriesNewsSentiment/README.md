TimeSeriesNewsSentiment is a python-coded tool that analyses the News Sentiment of the top 100 Google News related to a QUERY for each day given a TIME PERIOD.

It could be deployed in equity research, especially for the buy-side, to conduct primary research and mining for alternative data.

Key functions include:
1) Top news and sentiment scraping (returns a pandas dataframe)
2) Smoothing average of sentiment
3) Correlation analysis of sentiment vs asset return
4) Correlation analysis of Number of News vs High-Low (Intraday Volatility)
