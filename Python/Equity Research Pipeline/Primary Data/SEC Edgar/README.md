This project (SEC Edgar) is made with reference to a tutorial series by Quantburn https://www.youtube.com/@Quantburn

SEC Edgar is optimal for retreiving and parsing the data from the SEC immediately and downloads everything into an excel file once the company releases the financial statements so that there can be a rapid information edge for the buyside or sellside investment firm.

edgar_operations.py is the function code storer while the demo notebook is for usage illustration and common usage pipeline only.
